# ``SwiftDocker``

A Swift client for the Docker Engine API with async/await support and automatic Unix socket transport.

## Overview

SwiftDocker wraps the [Docker Engine API v1.53](https://docs.docker.com/engine/api/v1.53/) using code generated by [Swift OpenAPI Generator](https://github.com/apple/swift-openapi-generator). It provides:

- **Automatic connection** — reads `DOCKER_HOST` from the environment and picks the right transport, just like the Docker CLI.
- **Unix socket support** — communicates with a local daemon over a POSIX Unix-domain socket without requiring any third-party networking library.
- **Registry authentication** — injects `X-Registry-Auth` headers for push/pull operations against private registries.
- **Multiplexed log decoding** — parses Docker's framed stdout/stderr stream format so you get clean text output.

The raw OpenAPI-generated ``Client`` is always accessible at ``Docker/client``, giving you access to every Docker Engine endpoint directly.
